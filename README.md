## Problem
Manual inventory management causes stockouts, overstocking, and delays.

## Solution
An Oracle PL/SQL system that monitors inventory levels and automatically
generates purchase orders when stock falls below reorder thresholds.

## Features
- Inventory tracking
- Auto-reorder procedures
- Business rule enforcement via triggers
- Auditing of restricted actions
- BI-ready analytics queries
